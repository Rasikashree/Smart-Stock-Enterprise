💡 SMART LOCKING - THE CORE INNOVATION
═══════════════════════════════════════════════════════════════════════════════

HOW IT WORKS:

1. USER ADDS ITEM
   ↓
2. BACKEND CHECKS STOCK
   ↓
3. STOCK < 10?
   ├─ YES → ATOMIC LOCK:
   │        • Decrement stock immediately
   │        • Create cart_lock record
   │        • Set expiry = NOW() + 7 min
   │        • Return lockId to frontend
   │
   └─ NO  → NORMAL ADD:
            • Frontend handles in localStorage
            • No lock created
            • Fast checkout

4. FRONTEND COUNTDOWN
   • Timer counts down from 7:00
   • Shows "6:24 remaining"
   • "Extend" button available
   • "Remove" clears the lock

5. TIMER HITS 00:00
   • Frontend removes from display
   
6. CRON JOB (every 60 sec)
   • Finds expired locks
   • Returns stock to product
   • Marks lock as 'expired'
   • Logs to audit_audit

RESULT:
✓ No over-selling (atomic @Transactional)
✓ Fair distribution (first-come-first-served)
✓ Automatic cleanup (CRON)
✓ Full audit trail
✓ Zero manual intervention
